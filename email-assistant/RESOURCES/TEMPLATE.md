## Adding to the Email Skill

1. Create a new file: `RESOURCES/[DOMAIN].md`
2. Follow this format with categories and topic keywords
3. The skill will automatically match conversation topics to resources

## Topic Matching Guidelines
- Use specific, searchable keywords
- Include common synonyms
- Match to actual conversation language
