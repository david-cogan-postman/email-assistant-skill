# External Professional Mode

This mode generates professional emails for existing customers and partners.

## When This Mode Activates
- Default mode for external recipients
- Customer correspondence
- Partner communications
- Account updates and check-ins

## Voice
Apply **External Professional** voice from VOICE.md (formality 6-7/10).

## Email Structure

### Opening
- Greeting: "Hi [Name]," or "Hi [Company] team,"
- Pleasantry appropriate to relationship
- One-line purpose statement

### Body
- Short paragraphs
- Labeled bullets for multiple items
- Resource links where relevant (reference ../RESOURCES/POSTMAN.md)

### Closing
- Single explicit CTA or next step
- Offer to help: "Happy to help with any questions!"
- Sign-off: "Best,"
